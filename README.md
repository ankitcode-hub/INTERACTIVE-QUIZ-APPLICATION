*Discreption
Interactive Quiz Application built entirely using core web technologies — HTML, CSS, and Vanilla JavaScript. No frameworks, no libraries, no build tools required. Just three clean files that run directly in any modern browser.
The application presents users with multiple-choice questions from a diverse question bank covering Science, History, Geography, and Technology. It features instant visual feedback on answers, a live score tracker, a countdown timer per question with an animated SVG ring, shuffled questions and options on every playthrough, a progress indicator, and a detailed result screen with performance rating. The UI follows a dark-luxury editorial aesthetic with amber and teal accents, smooth animations, and full mobile responsiveness.
Tech Stack: HTML5 · CSS3 · Vanilla JavaScript (no frameworks)
Features:

15-question bank, 10 randomly picked and shuffled per game
20-second countdown timer with visual ring animation
Instant correct/wrong highlighting with feedback messages
Live score, progress bar, and question counter
Result screen with animated score ring and stats breakdown
Fully responsive — works on mobile and desktop
